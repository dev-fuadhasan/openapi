# Frontend - Open API Exposure Scanner

React + Vite + TailwindCSS frontend application.

## Environment Variables

Set `VITE_API_URL` in Netlify dashboard with your Cloudflare Worker URL.

Example: `https://open-api-exposure-scanner.xxxxx.workers.dev`

## Build

```bash
npm install
npm run build
```

Output will be in `dist/` directory.
